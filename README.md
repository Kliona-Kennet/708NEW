# Lost and Found App

## Project Overview
The Lost and Found App is an Android application developed using Kotlin Jetpack Compose and SQLite database in Android Studio. The application allows users to create, view, search, and remove lost or found item adverts.

## Features
- Create lost or found adverts
- Upload item images from gallery
- Store advert details using SQLite database
- View all saved adverts
- Filter adverts by category
- Display advert details
- Remove adverts from database
- Modern and simple Jetpack Compose UI

## Technologies Used
- Kotlin
- Jetpack Compose
- SQLite Database
- Android Studio
- Coil Image Library

## App Screens
- Home Screen
- Create Advert Screen
- Lost & Found Items List Screen
- Advert Details Screen

## Functionalities
- Form validation for empty fields
- Image upload support
- Date picker functionality
- Timestamp generation
- Category filtering
- Navigation between screens

## How to Run the Project
1. Open the project in Android Studio
2. Sync Gradle files
3. Run the application using an Android emulator or physical device
